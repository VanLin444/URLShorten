# URLShorten

Простое веб-приложение для сокращения ссылок на PHP.<br>
Пользователь вводит длинный URL, после чего приложение генерирует короткую ссылку с уникальным токеном.При переходе по сокращенной ссылке происходит перенаправление на исходный адрес.

## Функциональность
- Создание коротких ссылок
- Генерация уникальных токенов длиной от 6 до 10 символов
- Проверка корректности URL
- Защита от создания дубликатов ссылок
- Перенаправление по короткой ссылке
- Хранение данных в MySQL
